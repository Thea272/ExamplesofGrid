Los Ejemplos de Grid

Example 1

1. To define a Grid use display:grid or display:inline-grid on the parent element. You can then create a grid using the grid-template-columns and grid-template-rows properties. I am using the grid-gap property to create a gap between my columns and rows of 10px. This property is a shorthand for grid-column-gap and grid-row-gap so you can set these values individually. All direct children of the parent now become grid items and the auto-placement algorithm lays them out, one in each grid cell. Creating extra rows as needed.

1. Para definir una cuadrícula, utiliza display:grid o display:inline-grid en el elemento padre. Luego, puedes crear una cuadrícula usando las propiedades grid-template-columns y grid-template-rows.Estoy utilizando la propiedad grid-gap para crear un espacio entre mis columnas y filas de 10px. Esta propiedad es una forma abreviada de grid-column-gap y grid-row-gap, por lo que puedes establecer estos valores individualmente.Todos los hijos directos del padre ahora se convierten en elementos de la cuadrícula y el algoritmo de colocación automática los distribuye, uno en cada celda de la cuadrícula, creando filas adicionales según sea necesario.

Example 5

5. Grid auto-flow column The default behavior of Grid Auto Flow is to layout the elements by row, working along the row until there are no more slots then moving on to the next row. If a row is not declared then an implicit grid track will be created to hold the items. You can change this behavior by using the grid-auto-flow property. The default value is row but you can also specify column. The elements will then be laid out column by column, adding new columns if needed.

5. Flujo automático de columnas en cuadrículaEl comportamiento predeterminado del Flujo Automático de Cuadrícula es organizar los elementos en filas, trabajando a lo largo de la fila hasta que no haya más espacios y luego pasando a la siguiente fila. Si no se declara una fila, se creará una pista de cuadrícula implícita para contener los elementos.Puedes cambiar este comportamiento utilizando la propiedad grid-auto-flow. El valor predeterminado es fila, pero también puedes especificar columna. Los elementos se organizarán entonces columna por columna, añadiendo nuevas columnas si es necesario.

6. Multiple tracks in a track-list with auto-fill and minmax() The repeat syntax can take a track-list rather than a single value, this means you can repeat a more complex track-list. You can use minmax() in this track list, here I am making the max of my larger tracks 2fr, and the smaller ones 1fr. This means that the larger tracks will have more of any leftover space assigned.Currently requires Chrome Canary

6. Múltiples pistas en una lista de pistas con auto-fill y minmax() La sintaxis de repetición puede tomar una lista de pistas en lugar de un solo valor, lo que significa que puedes repetir una lista de pistas más compleja. Puedes usar minmax() en esta lista de pistas, aquí estoy haciendo que el máximo de mis pistas más grandes sea 2fr, y las más pequeñas 1fr. Esto significa que las pistas más grandes tendrán más del espacio sobrante asignado.Actualmente requiere Chrome Canary

7. Subgrid on columns and rows To define a subgrid in both dimensions - columns and rows - use the value subgrid on a grid item of the parent as the value of grid-template-columns and grid-template-rows. The nested grid is now tied to the number of grid tracks and dimensions of the parent grid.

7. Subred en columnas y filas Para definir una subred en ambas dimensiones - columnas y filas - utiliza el valor subred en un elemento de cuadrícula del padre como el valor de grid-template-columns y grid-template-rows. La cuadrícula anidada ahora está vinculada al número de pistas de cuadrícula y dimensiones de la cuadrícula padre.

Example 8

8. Aligning the grid with space-around and space-between. In this example I am creating a four column track grid, the tracks have absolute sizes and in total are smaller than the area of the grid container. This time I have used: align-content: space-around justify-content: space-between. This means that extra space is distributed around the tracks and our desired 10 pixel gutter gets more space. However so do any areas which span more than one track as where they cross a gutter they gain this extra space too.

8. Alineando la cuadrícula con espacio-around y espacio-between. En este ejemplo estoy creando una cuadrícula de cuatro columnas, las pistas tienen tamaños absolutos y en total son más pequeñas que el área del contenedor de la cuadrícula. Esta vez he utilizado: align-content: space-around justify-content: space-between Esto significa que el espacio extra se distribuye alrededor de las pistas y nuestro deseado margen de 10 píxeles recibe más espacio. Sin embargo, también lo hacen las áreas que abarcan más de una pista, ya que donde cruzan un margen, también ganan este espacio extra.
